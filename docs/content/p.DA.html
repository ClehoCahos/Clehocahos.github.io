            |                   IR_live                         |
            |                                                   |
            |   Le projet IR_live met Ableton au centre         |
            |   de la scène. Avec Live 12.2, Push II et un      |
            |   contrôleur MIDI, le but est tout simplement de  |
            |   "jouer de l'ordinateur".                        |
            |   Chaque live est une jam en direct, une réaction |
            |   à ce que propose le système.                    |
            |   Jouer un live IR revient souvent à jeter des    |
            |   dés, le tempo et les genres varient. Le voyage  |
            |   est avant tout temporel, il s'inscrit dans le   |
            |   réel.                                           |
            |   J'essaie d'utiliser les effets natifs           |
            |   d'Ableton le plus possible + des plugins        |
            |   faits maison, most of it is operator and MDD    |
